Where do i start?

I can tell you a great place to start. If you are completely new to HTML this is how you get started.

  1. Open up your Notepad and save the file as HelloWorld.html
  2. Copy paste the following code:
       < html >
          < head >
           < title>My Page < /title >
          < /head >
          < body>
            < h1> HELLO WORLD!!!< /h1>
          < /body>
      < / html>
  3. Save the file again. Click 'ctrl-save'
  4. Right click the file, open it in a web browser and see the magic happen.

A browser titled "My Page" opens up and gives you the message "Hello World!"

YOU DID IT! I really hope you did. And I am pretty sure you loved it!



This is the easiest way to break the ice and jump into coding..Go pat yourself! You just wrote a piece of code! Here comes an explanation to what you just did.

Tag Tags Used Explanation
1 < html> < /html> Defines that what we have inside the document is an HTML document
2 < head> < /head> Defines information about the document
3 < title> < /title> Defines a title for the document
4 < body> < /body> Defines the document's body
5 < h1> < /h1> Defines a heading


The above table should give you a rough idea on the code written above. All HTML document has a head and a body and all of it is defined inside a opening ['<'] and a closing tag ['/>']. All it takes is just 5 tags to create an html file and it opens up in your very own web browser, no server required.
